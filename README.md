# Travel Website for Erica

Travel website for Erica, a travel blogger and social media influencer. 

## Pages

- **Home** – Introduction and featured destination
- **Destinations** – Santorini, Kyoto, Cape Town
- **Travel Blog** – Blog posts with travel tips
- **Gallery** – Photo grid
- **Social Media** – Links to Erica's social channels

## Structure

```
travel-website-erica/
├── index.html
├── destinations.html
├── blog.html
├── gallery.html
├── social.html
├── css/
│   └── style.css
└── images/
```

## Images

Add your destination images to the `images/` folder. Filenames expected:

- `hero.jpg`
- `featured.jpg`
- `santorini.jpg`, `kyoto.jpg`, `capetown.jpg`
- `blog1.jpg`, `blog2.jpg`, `blog3.jpg`
- `gallery1.jpg` – `gallery8.jpg`
